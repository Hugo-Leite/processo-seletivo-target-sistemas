Você está em uma sala com três interruptores, cada um conectado a uma lâmpada em salas diferentes. Você não pode ver as lâmpadas da sala em que está, mas pode ligar e desligar os interruptores quantas vezes quiser. Seu objetivo é descobrir qual interruptor controla qual lâmpada. Como você faria para descobrir, usando apenas duas idas até uma das salas das lâmpadas, qual interruptor controla cada lâmpada?  

1- Ligar o interruptor A por um tempo, por exemplo 5 minutos ou mais.
2- Ligar o interruptor B
3- Desligar o interruptor A
4- Ir até uma sala. Hipoteticamente a sala A
5- Analisar a lâmpada. Se a lâmpada estiver acessa, ela é controlada pelo interruptor B, se estiver apagada mas estiver quente é controlada pelo interruptor A e se estiver apagada e fria é controlada pelo interruptor C.

Suponhamos que essa lâmpada estava acessa, ou seja ela é controlada pelo interruptor B

6- Volto para sala dos interruptores
7- Ligo o Interruptor A
8- Vou para outra sala. Hipoteticamente a sala B
9- Analisar a lâmpada. Se a lâmpada estiver acessa, ela é controlada pelo interruptor A, se estiver apagada é controlada pelo interruptor C.

Suponhamos que essa lâmpada estava apagada, ou seja ela é controlada pelo interruptor C

Sala A = Interruptor B
Sala B = Interruptor C
Sala C = Interruptor A
